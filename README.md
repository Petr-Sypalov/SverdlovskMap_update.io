# Туристическая карта Свердловской области (обновлённая)
URL-адрес: https://petr-sypalov.github.io/SverdlovskMap_update.io/ Данный репозиторий посвящён созданию туристической карты Свердловской области. Структура сайта представляет собой три основных раздела: первый - маршрут перевала Дятлова, второй - карта основных живописных мест и архитектруных сооружений, третий - описание состава участников группы Дятлова. Цель создаваемого сайта - помочь потенциальным туристам при выборе ключевых достопримечательностей Свердловской области.
